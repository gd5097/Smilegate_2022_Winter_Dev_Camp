# Smilegate_2022_Winter_Dev_Camp
스마일 게이트 2022 윈터 데브 캠프를 위한 레포입니다.

# 프로젝트에 대한 개요
URL을 입력창에 넣은 후 제출을 하면, 길이가 단축된 URL이 생성됩니다.
해당 URL을 사용할 경우 원본 URL링크와 같은 페이지로 이동하게 됩니다.

# 기술 스택
Python3

# 코드 중 확인 받고 싶은 부분
시간 관계상 완성에 목적을 둔 프로젝트인만큼 따로 확인 받고 싶은 부분은 없습니다.
